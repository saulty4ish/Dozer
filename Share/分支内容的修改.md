创建并推送分支



clone github上的仓库（ssh地址）

git clone [git@github.com](mailto:git@github.com):Credink/CTF-WP.git



cd进目录

cd CTF-WP



新建分支

git branch test



分支推送到github上去

git push origin test



------



修改分支内容并提交



切换分支

git checkout test



......各种修改......



add各种需要add的文件

git add ...



commit修改

git commit 



push提交到github上去

git push -u origin test



------



合并分支（Web）

git checkout master        //先切换到master

git merge Web



删除分支（Web）

git branch -d Web
